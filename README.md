# Pet Doc Finder

## Summary
This web app helps people to find the prices of veterinarian services around a location of their choice.

## Technology Used
* HTML/CSS/jQuery
* Node
* Express
* ReactJS
* Passport (authentication)
* Mocha/Chai (testing)
* Travis CI (integration)
* Deployed with Netlify
